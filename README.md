# .nvim

## Dependencies

- `git`, `make`, `unzip`, `gcc`, `ripgrep`, `tree-sitter`, `tree-sitter-cli`

## Linux & Mac Installation

```sh
git clone git@github.com:adriengivry/.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

## Windows Installation
If you're using `cmd.exe`:

```
git clone git@github.com:adriengivry/.nvim.git "%localappdata%\nvim"
```

If you're using `powershell.exe`

```
git clone git@github.com:adriengivry/.nvim.git "${env:LOCALAPPDATA}\nvim"
```
